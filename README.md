## USAGE
Get package source:
```
mkdir devapt-app-testbus
cd devapt-app-testbus
git init
git remote add origin https://github.com/lucbories/devapt-app-testbus.git
git pull origin master
```


Edit "start_node.cmd".


Get dependencies packages and run application:
```
cd devapt-app-testbus
npm install
npm start
```
